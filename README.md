# Personal Life Management System
A modular C project to manage Expenses, Habits, Health Records, Study Logs, and Reminders.


# Overview
This is a complete life-management system developed using **C Programming**.  
The project demonstrates:
- Modular Programming  
- File Handling  
- Time & Date Functions  
- Menu-Driven Structure  


# Folder Structure (Required)

The GitHub repository must contain:
/src → All .c files
/include → All .h files
/docs → ProjectReport.pdf
README.md → Project description
Example:

YourRepo/
```
├── src/
│ main.c
│ expense.c
│ habit.c
│ health.c
│ study.c
│ reminder.c
│
├── include/
│ expense.h
│ habit.h
│ health.h
│ study.h
│ reminder.h
│
├── docs/
│ ProjectReport.pdf
│
└── README.md
```
# Modules in the Project

**1. Expense Tracker**
- Add expense  
- View all expenses  
- Monthly summary  

**2. Habit Tracker**
- Add habit  
- Mark habit completed  
- View habits  

**3. Health Tracker**
- Add steps + calories  
- View health records  
- Today's health summary  

**4. Study Tracker**
- Add study session  
- View study entries  
- Today's study summary  

**5. Reminder System**
- Add reminder  
- View reminders  


How to Compile
Use GCC:
gcc src/*.c -I include -o main



How to Run

./main
